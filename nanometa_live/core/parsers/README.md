# Parsers Module

Comprehensive parsers for nanometanf v1.1.0 pipeline outputs.

## Overview

This module provides the `NanometanfOutputParser` class and `RealtimeMonitor` class for parsing all output formats generated by the nanometanf pipeline, including:

- **MultiQC Reports**: Aggregated quality control metrics
- **Kraken2 Classification**: Taxonomic assignment results
- **FASTP Quality Control**: Read filtering and quality statistics
- **BLAST Validation**: Species confirmation results
- **Real-time Batch Statistics**: Live processing metrics

## Quick Start

```python
from nanometa_live.core.parsers import NanometanfOutputParser

# Initialize parser
parser = NanometanfOutputParser("/path/to/nanometanf/results")

# Get top species
top_species = parser.get_top_species(n=10)

# Get classification summary
summary = parser.get_classification_summary()
print(f"Classification rate: {summary['overall']['classification_rate']:.2%}")
```

## Classes

### NanometanfOutputParser

Main parser class with 20+ methods for accessing pipeline outputs.

**Key Methods:**
- `get_top_species(n)` - Top N species by read count
- `get_classification_summary()` - Overall classification statistics
- `get_fastp_summary()` - Quality control summary
- `get_cumulative_stats()` - Real-time batch statistics
- `parse_kraken_report(sample)` - Sample-level Kraken2 data
- `parse_fastp_report(sample)` - Sample-level FASTP data

### RealtimeMonitor

Background monitoring for real-time batch statistics.

**Usage:**
```python
from nanometa_live.core.parsers import RealtimeMonitor

def on_new_batch(batch_data):
    print(f"New batch: {batch_data['reads_in_batch']} reads")

monitor = RealtimeMonitor("/path/to/results", on_new_batch)
monitor.start_monitoring(interval=10)
```

## Features

**Robust Error Handling:**
- Returns empty DataFrames/dicts on missing files (no exceptions)
- Retry logic for file watching in real-time mode
- Comprehensive logging for debugging

**Multiple File Format Support:**
- TSV parsing for MultiQC data
- JSON parsing for FASTP and batch statistics
- Kraken2 report format (`.kraken2.report.txt`)
- BLAST outfmt 6 (tabular)

**Flexible File Discovery:**
- Supports multiple naming conventions
- Automatic file pattern matching
- Graceful fallbacks for missing data

## Integration with Nanometa Live

The parser is designed for seamless integration with Dash dashboards:

```python
from dash import Input, Output
from nanometa_live.core.parsers import NanometanfOutputParser

parser = NanometanfOutputParser("/path/to/results")

@app.callback(
    Output('classification-rate', 'children'),
    Input('interval-update', 'n_intervals')
)
def update_classification_rate(n):
    summary = parser.get_classification_summary()
    rate = summary['overall']['classification_rate']
    return f"{rate:.2%}"
```

## Performance

- **On-demand parsing**: Files are read only when requested
- **No automatic caching**: Cache DataFrames in your application if needed
- **Background monitoring**: Real-time monitor runs in separate thread
- **Efficient data structures**: Uses pandas for large datasets

## File Formats

### Expected Directory Structure

```
results/
├── multiqc/multiqc_data/
├── fastp/
├── kraken2/
├── blast/
└── realtime_batch_stats/
```

### Supported File Patterns

**Kraken2:**
- `{sample}.kraken2.report.txt`
- `{sample}.cumulative.kraken2.report.txt`

**FASTP:**
- `{sample}.fastp.json`
- `{sample}_fastp.json`

**BLAST:**
- `{sample}_{taxid}.blast.txt`
- `{sample}_blast.txt`

**Batch Stats:**
- `batch_{number:04d}_stats.json`

## API Summary

| Category | Methods |
|----------|---------|
| MultiQC | `parse_multiqc_general_stats()`, `parse_multiqc_json()`, `parse_multiqc_fastp_data()`, `parse_multiqc_kraken_data()` |
| Kraken2 | `parse_kraken_report()`, `combine_kraken_reports()`, `get_top_species()`, `get_species_read_counts()` |
| FASTP | `parse_fastp_report()`, `combine_fastp_reports()`, `get_fastp_summary()` |
| BLAST | `parse_blast_results()`, `get_blast_validation_summary()` |
| Real-time | `parse_batch_stats()`, `get_latest_batch_number()`, `parse_all_batch_stats()`, `get_cumulative_stats()` |
| Summary | `get_classification_summary()` |
| Utility | `file_exists_with_retry()` |
