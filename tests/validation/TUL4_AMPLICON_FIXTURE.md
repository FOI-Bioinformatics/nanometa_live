# TUL4 Amplicon Test Fixture

## Overview

The barcode05 test fixture simulates a single-copy gene amplicon validation for *Francisella tularensis* (taxid 263) using the published TUL4 primer pair.

## Primer Details

**TUL4 Primers** (single-copy rRNA operon target):
- Forward (TUL4-435): 5' - GCTGTATCATCATTTAATAAACTGCTG - 3' (28 bp)
- Reverse (TUL4-863): 5' - TTGGGAAGCTTGTATCATGGCACT - 3' (24 bp)
- Target region: ~435–863 bp on the reference genome (~428 bp amplicon)

## Test Data Contents

### Kraken2 Classification (barcode05)
- **Total reads**: 7,000
- **F. tularensis**: 3,800 reads (54.3%)
- **Background flora**: C. acnes and B. subtilis (~2% each)

### Validation Data
- **Method**: minimap2 (genome-centric depth coverage)
- **PAF file**: `validation/minimap2/barcode05_taxid263.paf`
  - 400 synthetic reads mapping to the TUL4 amplicon window
  - Reads concentrated in chr positions 435–863
  - MapQ: 30–60, identity: 92–99%
- **Stats file**: `validation/minimap2/barcode05_taxid263.minimap2_stats.json`
  - Mapped reads: 3,650 / 3,800 (96.1% hit rate)
  - Avg identity: 97.8%, avg coverage: 0.94, avg MapQ: 56.0

### Reference Genome
- **Organism**: *Francisella tularensis* (taxid 263)
- **Simulated reference**: 1,892,775 bp (realistic genome size)
- **Contig name**: `Francisella_tularensis_chr`

## Contrast with Multi-Copy Genes

| Property | 16S Amplicon (barcode01) | TUL4 Amplicon (barcode05) |
|----------|-------------------------|-------------------------|
| **Copy number** | Multi-copy (~5–7) | Single-copy |
| **Genome size** | 4.4 Mb (M. tuberculosis) | 1.9 Mb (F. tularensis) |
| **Validation method** | Minimap2 (coverage depth) | Minimap2 (coverage depth) |
| **Per-batch drill-down** | Yes (batches 1–2) | No (cumulative only) |
| **Use case** | Incremental realtime validation | Final comprehensive check |

## Usage

The TUL4 fixture is automatically generated by `generate_all_synthetic_data()` in `tests/validation/generate_synthetic_data.py`. It is used in:
- Validation parser tests
- Coverage depth analysis tests
- Single-copy amplicon edge case validation

## Notes

- The amplicon is NOT included in per-batch drill-down (`batch/` directory) because single-copy genes typically do not benefit from incremental batch-by-batch validation—they are confirmed or not in a single pass.
- Unlike the multi-copy 16S rRNA, the narrow amplicon window (428 bp) allows for precise depth-of-coverage metrics without cross-locus ambiguity.
- The realistic 96.1% hit rate reflects high-quality single-copy amplicon targeting with minimal off-target mapping.
